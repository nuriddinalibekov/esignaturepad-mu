# esignaturepad-mu
Project Title: E-Signature Drawing Module: Programming Internet of Things University: Millat Umidi University Lecturer: Lazizbek Yusupov Semester: Fall 2025 This project is a touchscreen-based drawing pad built using an ESP8266 and a TFT display with touch support. It allows users to draw freehand lines on the screen using their finger or stylus.
